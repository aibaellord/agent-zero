## Tool: heisenberg_ultimate

Access the complete Heisenberg Singularity system - the unified orchestration layer for all cognitive systems.

### Actions:

1. **status** - Get comprehensive system status

```json
{
  "tool_name": "heisenberg_ultimate",
  "tool_args": {
    "action": "status"
  }
}
```

2. **process** - Process a task through optimal systems

```json
{
  "tool_name": "heisenberg_ultimate",
  "tool_args": {
    "action": "process",
    "task": "Analyze this problem using all available reasoning",
    "strategy": "parallel"
  }
}
```

Strategies:

- `auto`: Automatically select best approach
- `parallel`: Process simultaneously across systems
- `sequential`: Process through systems in order
- `consensus`: Get agreement from multiple systems
- `cascade`: Try systems until one succeeds

3. **health** - Perform comprehensive health check

```json

    "action": "health"
  }
}
```

4. **systems** - List all available systems by category

```json
{
  "tool_name": "heisenberg_ultimate",

    "action": "systems"
  }
}
```

5. **query** - Query a specific system

```json
{
  "tool_name": "heisenberg_ultimate",

```
